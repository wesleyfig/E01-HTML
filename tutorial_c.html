Estruturas de Repetição

Repetição com teste no início

Repete o bloco de operações enquanto a condição for verdadeira.

...
while (condição){
bloco de operações
}
...

Passos:

  1. avalia a condição:
    • se verdadeiro, executa o bloco de operações;
    • caso contrário, termina o laço.

Exemplo:

Faça um programa que mostre na tela os números de 1 a 100:

...
int numero = 1;
while (numero <= 100){
printf("%d", numero);
}
...

Observe que a variável numero é usada como um contador, ou seja, vai contar quantas vezes o loop será executado.

Repetição com teste no final

O comando do-while: é utilizado sempre que o bloco de comandos deve ser executado ao menos uma vez, mesmo que a condição seja falsa.

...
do {
bloco de operações
} while (condição)
...

Passos :

  1. executa o bloco de operações;
  2. avalia a condições:
    • se verdadeiro, re-executa o bloco de operações;
    • caso contrário, termina o laço.
Exemplo:

Faça um programa que exiba um menu ao usuário e receba uma opção entre 1 e 3:

int i;
do {
printf("Escolha uma opção:\n")
printf("(1) opção 1\n")
printf("(2) opção 2\n")
printf("(3) opção 3\n")
scanf("%d", &i);
} while ((i < 1) || (i > 3));

Repetição contada

...
for (inicialização; condição; incremento){
bloco de operações;
}
...
  1. Inicialização: inicia uma variável;
  2. avalia a condição. Se for verdadeira, executa o bloco de operações, senão encerra o laço;
  3. Incremento: ao término do bloco de operações, incrementa o valor da variável;
  4. repete o processo até que a condição seja falsa.

Passos

  1. Executa a inicialização;
  2. avalia a condição:
    • se verdadeiro, executa o bloco de operações;
    • caso contrário, termina o laço.
  3. Executa o incremento e volta para (2).

Em geral, utilizamos o comando for quando precisamos ir de um valor incial até um valor final;

Para tanto, utilizamos uma variável para realizar a contagem.

Exemplo:

Faça um programa que mostre na tela os números de 1 a 100:

...
int i;
for (i=1; i<=100; i++){
printf("%d \n", i);
}
...

Como sair prematuramente de um loop

  1. break: Força o loop terminar; Vai para a linha imediatamente depois do fim do loop.
  2. continue: Pula para a próxima iteração; Volta para o início do loop sem executar as instruções que estão abaixo do continue. O incremento é executado normalmente.
  3. exit: Finaliza o programa. Não deve ser utilizado a menos que você tenha certeza que a única solução é encerrar seu programa neste ponto.
